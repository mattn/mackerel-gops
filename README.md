# mackerel-gops

Upload [https://mackerel.io/](Mackerel) Custom Metrics from [gops](https://github.com/google/gops)

## Usage

```
$ mackerel-gops -p 12345 -name procname -service servicename
```

## Installation

```
$ go get github.com/mattn/mackerel-gops
```
